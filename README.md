# Платформа по продаже недвижимости Rentiz (учебный проект)

## Описание проекта 
Проект представляет собой веб-сайт по продаже недвижимости в различных странах. Главной целью этого проекта являлось отработатка и закрепление основных знание по **HTML** и **CSS** с применением **javascript** для реализации бургер-меню и слайдера. Макет реализован с использованием **Flexbox** и **CSS Grid** для точного позиционирования контента. Визуальная часть сайта использует препроцессор **SCSS** и методологию **БЭМ (BEM)** для структурирования стилей. Сайт полностью адаптивен благодаря использованию **media queries**, обеспечивая корректное отображение на всех типах устройств.
