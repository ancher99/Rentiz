# Платформа по продаже недвижимости Rentiz (учебный проект)

## **Описание проекта **
Проект представляет собой веб-сайт по продаже недвижимости в различных странах. Главной целью этого проекта являеться отработатка и закрепление основных знание по **HTML** и **CSS** с применением **javascript** для реализации бургер-меню и слайдера. В проекте были использованы **flex** и **grid** для выравнивания контента. Также был использовон препроцессор **SCSS**.
Проект выполнен с помощью **адаптивной верстки** на media запросах, для корректного и удобного отоброжения веб-сайта на различных устройств, по методологии **BEM**.
